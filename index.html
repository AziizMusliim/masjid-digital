📋 Panduan Setup

  1. Buka file README.md untuk panduan lengkap
  2. Setup Google Apps Script Backend terlebih dahulu
  3. Update URL Apps Script di file frontend-admin/js/api.js dan frontend-board/js/config.js
  4. Buka Admin Panel untuk mengelola konten
  5. Buka Papan Informasi di layar masjid
